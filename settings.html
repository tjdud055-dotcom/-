클라우드 동기화 (Supabase)

Supabase 프로젝트 URL과 anon key를 등록하면 그룹을 클라우드에 공유하고 실시간 동기화할 수 있어요.

책 검색 (카카오)

카카오 REST API 키를 등록하면 한국 책을 훨씬 정확하게 검색할 수 있어요.
키 없이도 Google Books로 검색되지만 표지·정확도가 낮을 수 있어요.

AI 토론 질문 (선택)

Anthropic API 키를 등록하면 그룹을 만들 때 AI가 책에 맞는 토론 질문을 자동 생성해줘요.
키는 이 기기의 localStorage에만 저장되며 외부로 전송되지 않습니다.
